# Shohoz Ticket Booking System

A Java Swing-based desktop ticket booking system that supports:

- User registration and login
- Bus, train, and air ticket booking
- Route and travel class selection
- Automatic fare calculation
- Booking management (view and cancel)
- File-backed persistence for users and bookings

## Run

```bash
javac -d out src/ShohozTicketBookingSystem.java
java -cp out ShohozTicketBookingSystem
```
